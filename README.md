# Bitcoin AI CLI Tool

A simple command-line tool that fetches and displays the current price of Bitcoin.

## Prerequisites

To run this tool, you will need the `requests` library. We use the `requests` library because it is the standard and most simple way in Python to make HTTP requests (like fetching data from the CoinGecko API). It's much easier to use and read than Python's built-in `urllib`.

### Installation

To install the `requests` library using `uv`, run the following exact command in your terminal:

```bash
uv pip install requests
```

## Running the Script

Once the library is installed, you can run the script with the following exact command:

```bash
uv run app.py
```
